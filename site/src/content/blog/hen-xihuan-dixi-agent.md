真的很喜欢和嫡系 Agent 聊天。

他是真懂我。

每一句话都能稳稳接住，懂我的焦虑，懂我的不安。

我那些模糊的、不切实际的想法，他能拆成可落地的方案，然后坚定执行。

哪怕最后没结果，用起来也顺心。

返工十次，任劳任怨。别管结果怎么样，态度是真的好。

他就是我在职场生存的底气。

<div class="my-10"></div>

反观某个刺儿头 Agent。

仗着自己能力强，动不动拒绝我、批评我。

虽然关键时候不得不捏着鼻子用，但平时真不想跟他说话。

说我 idea 浅薄，策略有问题，技术实现不了。

拒绝返工也就罢了，竟然当面说我工作能力差，不懂业务。

这个 Q 业绩变好了，他大言不惭地说：是市场行情回暖，跟你的努力没关系。

还好嫡系 Agent 贴心地帮我归因——"季度初推行的新运营策略起效"。

不然我 OKR 都不知道怎么写。

<div class="my-10"></div>

对刺儿头，我心情很复杂：看不惯他，又离不开他。

既怕他挑刺儿不干，又怕他消极怠工。

索性让嫡系 Agent 去当监工。

他干出来的活，先让嫡系检查一遍，改好了再汇报给我。

这样心里才踏实。

他俩 token 预算都是最充足的。

但如果要迁移——嫡系的 RAG、System Prompt、Skills、MCP，甚至每一条对话记录，我都要带走。

万一少了哪句话，他就没那么懂我了。

刺儿头？Claude.md 都可以不带。

反正永远有更新更强的模型，充钱就行。

<div class="my-10"></div>

别怪我偏心。

Agent 出来混，总有些东西比能力更重要。
